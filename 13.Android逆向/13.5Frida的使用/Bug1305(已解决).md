#### 2.准备工作

##### 电脑上安装frida
使用版本：
frida                   16.7.19
frida-tools             13.7.1

```shell
pip install frida==16.7.19, frida-tools==13.7.1
```

##### frida-server
手机：雷电模拟器 MI9(安卓9 x86_64) 开启文件root、系统可写入权限、ADB调试

第零：bug说明
```shell
# 刚开始使用的是frida-server-17.0.4会报错如下，
:/data/local/tmp # ./frida-server-17.0.4
{"type":"error","description":"ReferenceError: exports is not defined","stack":"ReferenceError: exports is not defined
    at /index.js:1:107","fileName":"/index.js","lineNumber":1,"columnNumber":107}
```

第一：下载 https://github.com/frida/frida/releases 找到对应frida版本的

使用版本：frida-server-16.7.19-android-x86_64.xz 解压后使用

frida-server-16.7.19-android-x86_64

第二：启动frida-server服务
```shell
# 查看手机的架构
C:\Users\H\Desktop>adb shell getprop ro.product.cpu.abi
x86_64

# 查看安卓手机正在运行的APP进程
C:\Users\H\Desktop>frida-ps -U


# 上传frida-server到/data/local/tmp目录下
C:\Users\H\Desktop>adb push frida-server-16.7.19-android-x86_64 /data/local/tmp
frida-server-16.7.19-android-x86_64: 1 file pushed, 0 skipped. 30.6 MB/s (99399448 bytes in 3.097s)

# 进入手机shell命令行
C:\Users\H\Desktop>adb shell
# 提权
gracelte:/ $ su
# 进入到/data/local/tmp目录中
:/ # cd /data/local/tmp
# 列出当前目录文件
:/data/local/tmp # ll
total 205176
-rw-rw-rw- 1 shell shell  99399448 2025-05-24 11:26 frida-server-16.7.19-android-x86_64
drwx--x--x 3 shell shell      4096 2025-05-23 15:10 oat
# 修改frida-server-16.7.19-android-x86_64文件权限
:/data/local/tmp # chmod 777 frida-server-16.7.19-android-x86_64
# 启动frida-server服务
:/data/local/tmp # ./frida-server-16.7.19-android-x86_64

```

第三：新开一个命令行并转发
```shell
C:\Users\H\Desktop>adb forward tcp:27043 tcp:27043
27043
C:\Users\H\Desktop>adb forward tcp:27042 tcp:27042
27042
```
