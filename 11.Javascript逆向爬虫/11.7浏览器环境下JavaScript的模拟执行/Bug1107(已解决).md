###### 报错
```shell
playwright._impl._errors.Error: Page.evaluate: TypeError: window.encrypts is not a function
    at eval (eval at evaluate (:234:30), <anonymous>:2:23)
    at UtilityScript.evaluate (<anonymous>:241:19)
```

###### 原因及解决方案
```python
# route方法中url参数：远程js的路径问题

page.route(
    "/js/chunk-10192a00.243cb8b7.js",
    lambda route: route.fulfill(path="./chunk.js")
)

# 书写完整的js路径前面加 **
page.route(
    "**/js/chunk-10192a00.243cb8b7.js",
    lambda route: route.fulfill(path="./chunk.js")
)
```
