#### AST
抽象语法树（Abstract Syntax Tree，AST）是源代码语法结构的一种抽象表示形式。它以树状结构展现编程语言的语法，每个节点代表源代码中的一种结构，例如变量声明、函数调用或控制流语句。AST 是代码解析和分析的核心工具，广泛应用于编译器、代码检查工具和代码转换工具中。

使用node构建环境，在一个纯净的目录中运行命令行命令
```shell
npm install -g @babel/node

npm init
npm install -D @babel/core @babel/cli @babel/preset-env

# 用babel包执行js代码使用下列模板
babel-node basic1.js
```
