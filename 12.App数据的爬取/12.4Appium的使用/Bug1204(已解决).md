### 1.准备工作

#### 安装 Appium 有两种方式
 第一种：https://github.com/appium/appium-desktop/releases 仓库中有安装包直接下载安装即可 

 使用版本 Appium-Server-GUI-windows-1.22.3-4.exe
 GUI可直接打开软件启动Appium服务即可(查看配置是否正确, 例如Android SDK的位置等环境变量)

 第二种：使用node.js安装
 ```shell
 # 安装appium 使用版本Appium v2.18.0
 npm install -g appium
     
 # 安装uiautomator2驱动 使用版本 - uiautomator2@4.2.3
 appium driver install uiautomator2
 
 # 启动Appium服务 服务地址 192.168.10.180:/wd/hub
 appium --address 192.168.10.180 --port 4723 --base-path /wd/hub
 ```

#### 下载appium-inspector
appium-inspector工具从appium-desktop分离出来的，现在要单独下载
使用版本 Appium-Inspector-2025.3.1-win-x64 便携版
https://github.com/appium/appium-inspector 仓库中有安装包直接下载即可

打开Appium-Inspector相关配置
```json
{
 "//": "ip地址端口和url路径",
 "address": "192.168.10.180",
 "port": "4723",
 "base-path": "/wd/hub"
}
```
```json
{
  "//": "手机和测试软件的相关配置",
  "platformName": "Android",
  "deviceName": "Redmi_K30_5G",
  "appPackage": "com.goldze.mvvmhabit",
  "appActivity": ".ui.MainActivity",
  "noReset": "True"
}
```
     

#### 使用手机为红米K30(MIUI13.0.5 安卓12) 通过数据线连接电脑
手机上安装测试软件 (https://app5.scrape.center/)
 ```shell
 # 使用adb命令查看手机连接电脑相关信息
 C:\Users\H>adb devices -l
 List of devices attached
 b5d2f0f  device product:picasso model:Redmi_K30_5G device:picasso transport_id:13
 ```
 **手机要打开开发者模式以及开发者模式中调试板块中的相关配置**
 (否则appium命令行报错见appium-server-logs.txt, appium-inspector中报错见appium_bug.png)
