###### 6.aiohttp的代理设置 SOCKS代理代码运行报错
```text
loop = loop or asyncio.get_running_loop()
                   ^^^^^^^^^^^^^^^^^^^^^^^^^^
RuntimeError: no running event loop

RuntimeError: no running event loop，原因是： ProxyConnector.from_url(...) 在模块顶层执行
该调用内部需要当前运行的事件循环 但此时事件循环尚未创建，所以报错

解决：
# 这段代码放在函数中声明
connector = ProxyConnector.from_url('socks5://127.0.0.1:7890')

```
